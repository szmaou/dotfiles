<h1 align="center">Simple Configuration Arch + Hyprland</h1>

### Stack

| Category         | Program                                                                                             |
| ---------------- | --------------------------------------------------------------------------------------------------- |
| WM               | [hyprland](https://github.com/hyprwm/hyprland)                                                      |
| Shell            | [fish](https://github.com/fish-shell/fish-shell) + [starship](https://github.com/starship/starship) |
| Terminal         | [kityy](https://github.com/kovidgoyal/kitty)                                                        |
| Bar              | [waybar](https://github.com/alexays/waybar)                                                         |
| Launcher         | [fuzzel](https://codeberg.org/dnkl/fuzzel)                                                          |
| Notifications    | [swaync](https://github.com/ErikReider/SwayNotificationCenter)                                      |
| OSD              | [swayosd](https://github.com/ErikReider/SwayOSD)                                                    |
| Lockscreen       | [hyprlock](https://github.com/hyprwm/hyprlock)                                                      |
| Session manager  | [uwsm](https://github.com/Vladimir-csp/uwsm)                                                        |
| Theme engine     | [matugen](https://github.com/InioX/matugen)                                                         |
| Wallpaper engine | [awww](https://codeberg.org/LGFae/awww) + [waypaper](https://github.com/anufrievroman/waypaper)     |
| Window switchers | [snappy-switcher](https://github.com/OpalAayan/snappy-switcher)                                     |
| File manager     | [thunar](https://gitlab.xfce.org/xfce/thunar) & [yazi](https://github.com/sxyazi/yazi)              |
